# LuminaCart AI - Intelligent Retail Assistant

🚀 **Winning Idea**: A real-time intelligent shopping cart system that personalizes the customer journey while optimizing retailer operations.

## 🌟 Overview
LuminaCart AI is a high-fidelity prototype using the **Aurora Obsidian** winning theme. It demonstrates a **Cloud-First** architecture designed for the Retail & E-commerce vertical.

## 🎨 Winning Theme: Aurora Obsidian
- **Obsidian Core**: Deep, high-contrast black backgrounds for premium focus.
- **Aurora Mesh Gradients**: Interactive, flowing gradients (Blue, Purple, Pink) that create a living interface.
- **Ultra-Glassmorphism**: 16px backdrop-blurs with Frosted Glass borders for a modern tech aesthetic.

## ☁️ Cloud-First Architecture (GCP)
The solution is designed to be hosted on **Google Cloud Platform**:
- **Vertex AI**: Powering the real-time recommendation inference engine.
- **Compute Engine**: Global low-latency edge nodes for kiosk and mobile synchronization.
- **Cloud Run**: Scalable microservices for the LuminaCart logic.
- **GCP Cloud Console**: A dedicated dashboard for monitoring infrastructure health and AI model drift in real-time.


## 🛠️ Tech Stack & Google Services
- **React + Vite**: High-performance frontend framework.
- **Tailwind CSS**: Modern utility-first styling for high-fidelity UI.
- **Google Fonts**: Custom typography (`Inter` & `Outfit`) for a premium corporate aesthetic.
- **Gemini-Inspired Logic**: Personalized insight text generated via a simulated Google GenAI interface.
- **Framer Motion**: State-of-the-art micro-animations for an "Advanced Agentic" feel.

## 🚀 How to Run (Full Stack)
To experience the full capability of the AI assistant, run both the backend and frontend:

1. **Start Backend**: `npm run server` (Runs on http://localhost:3002)
2. **Start Frontend**: `npm run dev` (Runs on http://localhost:5178)
3. **Open Browser**: Navigate to `http://localhost:5178`

## ⚙️ Core Logic & Approach
The solution uses a **Decoupled Architecture**:
1. **Node.js Express Backend**: Processes recommendation requests, manages inventory state, and simulates "Google Cloud" API responses.
2. **React Liquid Frontend**: A high-performance UI that fetches real-time insights from the backend for a truly 'working' assistant experience.
3. **Lumina Engine**: Heuristic affinity logic + Inventory awareness, framed with Gemini-inspired metadata.

## ♿ Accessibility & Quality
- Implemented with **ARIA roles** and semantic HTML for inclusivity.
- Modular architecture ensures the code remains clean, maintainable, and **highly optimized** (under 1MB).
